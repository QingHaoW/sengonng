一、什么是心梗？

心脏上有好多给心脏供血的血管，就像水管一样。一旦血管突然堵死，心脏没血没氧气，心肌就会坏死，这就是心肌梗死。

⚠️

说白了就是心脏血管堵死、心肌饿死了。发病快、死亡率高，千万别耽误！

🚨

二、心梗发作有啥信号？一定要记牢

  1. 胸口难受：胸口闷、压榨疼、像压了大石头，喘不上气，持续超过15–20分钟，休息也不好转。
  2. 放射性疼：不只是胸口，还会串到左肩、后背、脖子、下巴、上腹。
  3. 伴随症状：出大汗、恶心想吐、头晕心慌、浑身无力、濒死感。
  4. 特殊人群：老人、糖尿病、女性可能不胸痛，只觉得胃疼、胸闷、喘气费劲，千万别当成胃病。
🔔

记住：只要突然胸闷胸痛、大汗憋喘，先往心梗想，别硬扛！

🏥

三、一旦发作，立刻这么做

  1. 马上停下所有活动，平躺或半坐着休息，别走路、别上楼、别用力。
  2. 立刻打 120，别自己开车去医院，别等家人、别熬一会儿看看。
  3. 有速效救心丸、硝酸甘油可按说明含服，不确定血压不要乱吃药。
  4. 保持透气、放松情绪，不要喝水吃东西，防止呕吐呛到。

🕐 时间就是生命！发病后 120 分钟内开通血管，抢救成功率最高。

👥

四、哪些人容易得心梗？

  • 高血压、高血脂、糖尿病病人
  • 长期抽烟、喝酒、熬夜、压力大
  • 肥胖、不爱运动、爱吃油腻咸食
  • 有冠心病、心梗家族史的人
🛡️

五、日常怎么预防心梗（重点）

🥗 1. 吃的方面

少吃：肥肉、动物内脏、油炸、烧烤、咸菜、高糖饮料

多吃：蔬菜、水果、粗粮、豆制品、鱼肉

原则：少油、少盐、少糖，每餐七分饱，别暴饮暴食，尤其晚饭别吃太撑。

🏃 2. 生活习惯
  • 绝对戒烟，少喝酒，最好不喝
  • 不熬夜，每天睡够7小时，少生气、少激动，情绪别大起大落
  • 适度运动：散步、慢跑、打太极，别剧烈运动，饭后别马上动
💊 3. 慢病一定要管好

高血压、高血脂、糖尿病按时吃药，不能自己停药、减药，定期测血压、血糖、血脂，定期体检。

🚫 4. 记住两个"千万别"
  • 千万别胸痛硬扛、忍一忍
  • 千万别乱信偏方、不去正规医院
📋

六、出院后注意事项

  1. 按时吃药：医生开的药终身遵医嘱，不擅自停药换药。
  2. 避免诱因：避免劳累、受凉、情绪激动，大便别用力屏气。
  3. 定期复查：有任何胸闷、心慌、气短马上就医。

❤️ 出院不等于痊愈，坚持服药、定期复查才是长期健康的保障。
